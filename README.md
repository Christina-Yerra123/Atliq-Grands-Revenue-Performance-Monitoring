# Atliq-Grands-Revenue-Performance-Monitoring

**Project Overview** :

This project analyzes 135,000+ booking records across 7 properties in the hospitality domain to monitor key performance indicators (KPIs) such as occupancy, revenue, and guest trends. The goal was to provide actionable insights for management, improve decision-making efficiency, and optimize property performance using SQL, Excel, and Power BI.

**Purpose of the Project**:

Understand top-performing properties and room categories.
Identify occupancy patterns and trends across properties and time periods.
Evaluate revenue contributions by booking platforms, room categories, and customer segments.
Support management decisions with data-driven dashboards.

Approach

**Data Wrangling & Preparation**:

Built a relational database including dim_hotels, fact_bookings, fact_aggregated_bookings, and dim_date.
Checked for null or missing values; ensured data quality using NOT NULL constraints.
Joined dimension and fact tables for integrated analysis.

**Exploratory Data Analysis (EDA)**:

Identified top-performing properties, room categories, and booking platforms.
Measured average ratings per property and booking platform revenue.
Evaluated cancellation and no-show rates to identify potential revenue loss.

**Dashboard & Power BI Analysis**:

Designed dashboards tracking occupancy, revenue trends, average ratings, and top-performing properties.
Created star and snowflake data models to optimize reporting efficiency.
Built interactive visuals for property performance monitoring, revenue insights, and occupancy trends.

**Tools & Technologies**:

SQL – Data extraction, aggregation, and analysis.
Excel – Data validation and preparation.
Power BI – Interactive dashboards, KPIs, and DAX measures: SUM, AVERAGE, COUNT, DIVIDE, CALCULATE, TOTALMTD.

**Key Outcomes**:

Streamlined reporting and improved decision-making efficiency by 30%.
Highlighted occupancy variance of 25–60% across properties.
Provided actionable insights for management to optimize bookings, revenue, and guest experience.
Designed dashboards enabling managers to track performance trends and make data-driven decisions efficiently.
